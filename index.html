Tentang Saya

Halo! Saya Aditya Pratama W.P, seorang seniman 3D dan editor video dengan passion di animasi, texturing, dan editing. Saya selalu antusias untuk belajar dan menciptakan visual yang menarik.

Pengalaman saya meliputi penggunaan software seperti Blender, Substance Painter, Adobe Premiere Pro, After Effects. Saya telah mengerjakan proyek animasi pendek, texturing aset game, video promosi, dan anime music video.

Perlu diketahui bahwa untuk proyek texturing (Seluruh karakter dan animasi untuk proyek game praktik), model 3D disediakan oleh teman saya, dengan fokus utama saya pada kualitas texturing dan pengaplikasiannya. Sedangkan untuk sebagian besar proyek animasi 3D saya (Yang bukan untuk keperluan animasi game), aset mesh sering mengambil dari internet.
